An Arduino-based wearable glove that helps in emergency communication and safety monitoring.

## Hardware Components

* Arduino Nano
* Raspberry Pi Zero W
* OLED Display
* Push Buttons
* Sensors

## Description

This project implements a smart glove device designed to assist communication and safety.
The glove can trigger signals or actions through programmed gestures and button inputs using Arduino-based embedded control.
